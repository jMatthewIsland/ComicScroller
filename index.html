Description

Vertically Scrolling Comics is an app developed by Joe C. Bernstein, Ethan Coates, Matthew Island, and Meehan Imam that allows users build, share, and read comics.

Key Features

  • Enjoy a Dynamic Comic Reader with Scrollable Panels
  • Save Your Own Comics with Online Storage
  • Interact with Other Fans with User Comments
  • Share One Device with Multi-User Account Support
